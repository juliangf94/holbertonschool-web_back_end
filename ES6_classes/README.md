# ES6 Classes

## Description

This project covers ES6 class syntax in JavaScript. Each task implements a class with specific attributes, getters, setters, static methods, and inheritance patterns.

## Learning Objectives

- How to define a class
- How to add methods to a class
- Why and how to add a static method to a class
- How to extend a class from another
- Metaprogramming and symbols

## Requirements

- NodeJS 20.x
- All files use `.js` extension
- ESLint with airbnb-base config
- Jest for testing

## Setup

```bash
npm install
```

## Run files

```bash
npm run dev <filename>
```

## Test and lint

```bash
npm run test
npm run full-test
```

## Tasks

| File | Description |
|------|-------------|
| `0-classroom.js` | Basic class with constructor |
| `1-make_classrooms.js` | Function returning array of class instances |
| `2-hbtn_course.js` | Class with getters, setters and type validation |
| `3-currency.js` | Class with instance method |
| `4-pricing.js` | Class with static method and instanceof validation |
| `5-building.js` | Abstract class pattern using `new.target` |
| `6-sky_high.js` | Class inheritance with `extends` and `super` |
| `7-airport.js` | Custom `Symbol.toStringTag` |
| `8-hbtn_class.js` | Custom `Symbol.toPrimitive` |
| `9-hoisting.js` | Fix hoisting bugs in class declarations |
| `10-car.js` | Clone method using `Symbol.species` |
