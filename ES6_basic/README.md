# ES6 Basics

## Description

This project covers the fundamentals of ES6 (ECMAScript 2015) features in JavaScript. Each task focuses on a specific ES6 concept, rewriting older JavaScript patterns with modern syntax.

## Learning Objectives

- Understand what ES6 is and its new features
- Difference between `const`, `let`, and `var`
- Block scoping
- Arrow functions and default parameters
- Rest and spread syntax
- String template literals
- Object property shorthand and computed property names
- ES6 method properties
- `for...of` loops
- Iterators

## Requirements

- NodeJS 20.x
- All files use `.js` extension
- ESLint with airbnb-base config

## Setup

```bash
npm install
```

## Run files

```bash
npm run dev <filename>
# Example:
npm run dev 0-main.js
```

## Tasks

| File | Description |
|------|-------------|
| `0-constants.js` | Use `const` and `let` instead of `var` |
| `1-block-scoped.js` | Block scoping with `const` inside `if` blocks |
| `2-arrow.js` | Arrow functions with lexical `this` |
| `3-default-parameter.js` | Default parameter values |
| `4-rest-parameter.js` | Rest parameter syntax |
| `5-spread-operator.js` | Spread syntax to concatenate arrays and strings |
| `6-string-interpolation.js` | Template literals |
| `7-getBudgetObject.js` | Object property shorthand |
| `8-getBudgetCurrentYear.js` | Computed property names |
| `9-getFullBudget.js` | ES6 method properties |
| `10-loops.js` | `for...of` loops |
| `11-createEmployeesObject.js` | Dynamic object keys with computed properties |
| `12-createReportObject.js` | Object spread and method properties combined |
