# Python - Variable Annotations

Type annotations in Python using the `typing` module. Covers basic types (`int`, `float`, `str`, `bool`) and complex types (`List`, `Tuple`, `Dict`, `Union`, `Optional`, `Callable`, `Iterable`, `Sequence`).

Annotations are not enforced at runtime — they serve as hints for tools like `mypy` and for code readability.
