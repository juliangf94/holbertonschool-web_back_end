# ES6 Promises

## Description

This project covers JavaScript Promises and asynchronous programming in ES6. Tasks implement Promise creation, chaining, error handling with `try/catch`, and `async/await` syntax.

## Learning Objectives

- Promises: how, why, and what
- How to use the `then`, `resolve`, and `catch` methods
- How to use every method of the Promise object
- `throw` / `try`
- The `await` operator
- How to use an `async` function

## Requirements

- NodeJS 20.x
- All files use `.js` extension
- ESLint with airbnb-base config
- Jest for testing

## Setup

```bash
npm install
```

## Run files

```bash
npm run dev <filename>
```

## Test and lint

```bash
npm run test
npm run full-test
```
